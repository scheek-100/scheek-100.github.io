BUILD LOG: WEBSITE CUSTOMIZATION MILESTONES

A chronicle of how I evolved my site—from a basic Bootstrap template to a cyberpunk-themed tech portfolio that bridges design, engineering, and storytelling.

Folio Framework & Flask Routes Setup Process

Building the custom.css theme layer with neon accents

Testing responsive layouts for the Tech Portfolio section

Milestone 1: Launched the First Prototype

Started with the Folio Bootstrap framework and stripped out unused components. Built the site structure with index.html, main.css, and custom.js. 


              <section id="hero" class="hero">
                <h1>SASHA CHEEK</h1>
                <p>I am an Engineer</p>
              </section>

Milestone 2: Designed the Cyberpunk Theme

Replaced the default Bootstrap palette with neon cyan, magenta, and electric blue accents. Integrated Orbitron and DotGothic16 fonts to achieve a futuristic display aesthetic. 


              body {
              background: #12151f;
              color: #00ffff;
              font-family: 'DotGothic16', sans-serif;
              }

Milestone 3: Engineered Page Automation

Connected sections to load dynamically with AOS animations and reusable components. Built scroll animations, lazy loading, and used Swiper.js for portfolio carousels. 


              AOS.init({
                duration: 800,
                easing: 'ease-in-out',
                once: true
              });

Milestone 4: Published + Iterated

Deployed the site to production, refined the layout for mobile, and added new sections for the Tech Portfolio and Published Work.
